Совет
  • .col-* где * значение может быть { sm | md | lg | xl | xxl }
  • .col-# где # значение может быть от 1 до 12
  • .bg-{color} где color значение может быть { primary | secondary | success | danger | warning | info | light | dark }
  • Класс .bd-highlight предназначен только для демонстрационных целей
